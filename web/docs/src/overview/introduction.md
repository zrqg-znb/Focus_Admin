# 项目介绍

## 概述

**Focus Admin** 是一套企业级全栈管理系统解决方案，采用前后端分离架构设计。系统旨在提供一个功能完善、易于扩展的管理平台基础框架，涵盖项目管理、绩效管理、权限管理等核心业务功能。

## 项目特性

### 🚀 现代化技术栈

- **后端**: Django 4.x + Django Ninja + MySQL + Redis
- **前端**: Vue 3 + TypeScript + Element Plus + Pinia
- **工程化**: pnpm + Monorepo + Turbo

### 🔐 完善的权限体系

- 基于 RBAC 的权限管理模型
- 支持用户、角色、权限、菜单、部门的完整管理
- 前后端权限联动，支持按钮级权限控制

### 📦 模块化设计

- 清晰的模块划分，低耦合高内聚
- 核心模块与业务模块分离
- 支持快速扩展新业务模块

### 🛠️ 开发友好

- 完善的类型定义
- 统一的代码规范
- 丰富的工具函数

## 功能模块

### 核心模块 (Core)

| 模块 | 说明 |
| --- | --- |
| 认证模块 | 用户登录、JWT Token 管理、OAuth 登录 |
| 用户管理 | 用户增删改查、状态管理、密码重置 |
| 角色管理 | 角色定义、权限分配、角色继承 |
| 权限管理 | 细粒度权限控制、API 权限 |
| 菜单管理 | 动态菜单配置、路由管理 |
| 部门管理 | 组织架构树、部门层级 |
| 字典管理 | 数据字典维护、枚举配置 |

### 业务模块 (Apps)

| 模块 | 说明 |
| --- | --- |
| 项目管理 | 项目、迭代、里程碑、代码质量分析 |
| 绩效管理 | 绩效指标定义、数据导入、统计分析 |
| 代码合规 | 代码规范检查、合规报告 |
| 交付矩阵 | 交付进度跟踪、矩阵展示 |
| 集成报告 | 外部系统数据集成、报告生成 |

### 系统功能

| 功能 | 说明 |
| --- | --- |
| 任务调度 | 定时任务管理、APScheduler 集成 |
| 文件管理 | 文件上传下载、存储管理 |
| 日志管理 | 操作日志、登录日志记录 |
| 系统监控 | 服务器状态、数据库监控 |

## 项目结构

```
Focus_Admin/
├── backend-django/          # Django 后端
│   ├── application/         # 应用配置
│   ├── apps/               # 业务模块
│   ├── common/             # 公共模块
│   ├── core/               # 核心模块
│   ├── env/                # 环境配置
│   └── scheduler/          # 调度模块
│
└── web/                    # 前端 Monorepo
    ├── apps/web-ele/       # Element Plus 主应用
    ├── packages/           # 共享包
    │   ├── @core/          # 核心功能
    │   ├── effects/        # 副作用
    │   ├── stores/         # 状态管理
    │   └── ...
    └── docs/               # 项目文档
```
